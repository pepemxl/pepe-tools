# Video PWA

## Descripción

Esta es una aplicación web progresiva (PWA) que permite capturar imágenes desde la cámara del dispositivo y mostrarlas en un canvas.

## Características

- Captura de video desde la cámara
- Cambio entre cámara frontal y trasera
- Captura de imágenes en canvas
- PWA con service worker

## Instalación

1. Clona el repositorio
2. Copia la carpeta `video` a la carpeta `pepe_tools`
3. Ejecuta `python -m http.server` en la carpeta `pepe_tools`
4. Abre `http://localhost:8000/video/` en tu navegador

## Uso

1. Haz clic en "Iniciar Cámara"
2. Haz clic en "Capturar Imagen en Canvas"
3. La imagen capturada se mostrará en el canvas
4. Haz clic en "Cambiar Cámara" para cambiar entre cámara frontal y trasera
5. Haz clic en "Detener Cámara" para detener la cámara
